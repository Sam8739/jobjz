# 图片优化指南

## 🎯 目标
将网站中的PNG图片转换为WebP格式，减少文件大小50-80%，提升网站加载速度。

## 📊 当前问题
- 多个PNG图片超过500KB
- 总图片大小约15MB
- 影响网站加载速度

## 🛠️ 解决方案

### 方案1：使用Python脚本（推荐）

#### 步骤1：安装依赖
```bash
pip install Pillow
```

#### 步骤2：运行转换脚本
```bash
python convert_to_webp.py
```

#### 步骤3：更新HTML引用
```bash
python update_html_references.py
```

### 方案2：使用在线工具

#### 推荐工具
1. **Squoosh.app** (Google官方)
   - 访问：https://squoosh.app/
   - 拖拽图片到页面
   - 选择WebP格式，质量85%
   - 下载转换结果

2. **Convertio.co**
   - 访问：https://convertio.co/png-webp/
   - 批量上传PNG文件
   - 选择WebP格式
   - 下载所有转换结果

### 方案3：使用桌面软件

#### 推荐软件
- **ImageOptim** (Mac)
- **FileOptimizer** (Windows)
- **GIMP** (跨平台)

## 📋 操作步骤

### 1. 转换图片格式
```bash
# 运行转换脚本
python convert_to_webp.py
```

### 2. 检查转换结果
- 查看 `webp/` 目录
- 确认所有图片都已转换
- 检查图片质量是否满意

### 3. 移动WebP文件
```bash
# 将WebP文件移动到项目根目录
move webp\*.webp .
```

### 4. 更新HTML文件
```bash
# 运行HTML更新脚本
python update_html_references.py
```

### 5. 测试网站
- 在浏览器中打开网站
- 检查所有图片是否正常显示
- 测试不同设备和浏览器

### 6. 清理文件
```bash
# 删除备份文件（测试正常后）
del *.backup

# 可选：删除原始PNG文件（建议保留作为备用）
# del *.png
```

## ⚠️ 注意事项

### 浏览器兼容性
- **支持WebP**: Chrome 23+, Firefox 65+, Safari 14+, Edge 18+
- **不支持**: IE浏览器
- **建议**: 保留PNG作为备用格式

### 图片质量
- WebP质量设置为85%可保持良好视觉效果
- 可根据需要调整质量参数
- 建议在转换前备份原始文件

### 文件管理
- 转换前备份所有原始文件
- 测试确认无误后再删除PNG文件
- 保留原始文件作为备用

## 🔧 高级优化

### 响应式图片
```html
<picture>
  <source srcset="image.webp" type="image/webp">
  <source srcset="image.png" type="image/png">
  <img src="image.png" alt="描述">
</picture>
```

### 懒加载
```html
<img src="image.webp" alt="描述" loading="lazy">
```

### 预加载关键图片
```html
<link rel="preload" as="image" href="hero-image.webp">
```
